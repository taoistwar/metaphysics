### 零维（战术维）：把当下做到极致，美好自然呈现

零维，在几何学上就是一个点。点，是孤立的，是与外在世界隔绝的。

守着这一点，你将无处可去，也无处可退；你不知道机遇在哪里，也不知道风险会不会来，你只能守在这里。
守在这里干什么呢？每天都在研究战术（Tactic）问题：天上掉粮食，怎么才能多收一些？外面来敌人，怎么才能打败对手？
