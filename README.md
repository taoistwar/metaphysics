## 视角玄学
分工才能专注；专注才能精益求精。
### 0. 战术——极致匠人
把当下做到极致，美好自然呈现
### 1. 战略——纵向目标
不要用战术的勤奋掩盖战略的懒惰 
### 2. 模式——横向跨界
商业模式就是利益相关者的交易结构
如果说战略是一条条车道，那么，全局观就是凌空俯视所有车道的格局。
### 3. 创新——空间颠覆
颠覆式创新让不可能成为可能
空间,更高的高度突破颠覆
### 4. 时间——阶段调整
原因通常不在结果附近
发展的眼光看问题，万物皆有周期。
#### 5. 概率——多次尝试
正确的事情重复做
做有准备的人，每次尝试都会增加下一次成功的概率。
